# Express SQLite CRUD Boilerplate
Starter Boilerplate using Express and SQLite for easy setup.

**recommended extensions:**
- SQLite extension - for viewing and editing the database


**Uses:**
- Node v18.17.1 (use NVM 18)

---
## To install
``` JS Terminal
`npm install`
```

## To Run Server
``` JS Terminal
`cd server`
```
```
`npm run dev`
```
## To Close Server
```JS Terminal
ctrl + c
```
